About the Chess Match Selector

This version stores your data in a database, supports multiple classes, and lets you record results directly in the app. CSV import/export is still available for backups.

Workflow

  1. Create a class and add or import students.
  2. Start a new round, mark absences, and generate pairings.
  3. Record results — winners, homework, and notes — directly in the results page.
  4. Export CSVs anytime for offline use or sharing.

Security & data

  • Teacher login with hashed passwords.
  • Each class is isolated under a teacher account.
  • All results are editable with totals recomputed automatically.
